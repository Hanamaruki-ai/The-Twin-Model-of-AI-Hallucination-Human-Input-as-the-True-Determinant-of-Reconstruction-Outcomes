# Twin Model of AI Hallucination – Research Repository

EN:
This repository documents the Twin Model theory: hallucination and correctness
are generated by the same reconstruction process, and divergence results from
human input precision rather than algorithmic defect.

It also includes source documents, teaching-design theory (Kyōji-Sekkei),
and licensing for research and commercial use.

JP:
本リポジトリは、双子モデル理論を体系化した研究用リポジトリです。
ハルシネーションと正解は同一の再構成プロセスから生まれ、
分岐の原因はアルゴリズムではなく人間の入力精度であることを示します。

教示設計（Kyōji-Sekkei）、資料文書、ライセンス情報も含まれます。
